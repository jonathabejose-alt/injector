## Features

- **Real-time injection** — writes FFlags directly into Roblox memory
- **124+ presets** — FPS, Performance, Network, Fun, User Configs
- **Auto-update offsets** — fetches fresh offsets automatically on startup
- **Auto Apply** — continuously re-injects on a configurable interval
- **Kill Switch** — instantly reverts all injected flags
- **Injector Status** — live log with risk classification per flag
- **Custom themes** — background images, accent colors, shimmer title
- **Custom cursors** — 14 crosshair styles
- **Lag Switch** — network throttle tool
